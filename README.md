# 📊 Alura Store - Desafio 1

## 🎯 Objetivo
Este projeto faz parte do **Challenge Data Science 1 da Alura** e tem como objetivo analisar o desempenho das quatro lojas da **Alura Store**, identificando a unidade menos eficiente para ser vendida.  
A análise considera métricas como **faturamento total, categorias de produtos, avaliações dos clientes, produtos mais e menos vendidos e custo médio de frete**.

📈 Principais Resultados
Faturamento total por loja:

Loja 1: R$ 1.534.509,12

Loja 2: R$ 1.488.459,06

Loja 3: R$ 1.464.025,03

Loja 4: R$ 1.384.497,58

Média de avaliações: todas próximas de 4, com Loja 2 e Loja 3 ligeiramente melhores.

Frete médio: Loja 4 tem o menor custo de frete, mas não compensa o faturamento baixo.

✅ Conclusão
A Loja 4 apresenta o menor faturamento e é identificada como a unidade menos eficiente.
Portanto, a recomendação é que o Senhor João venda a Loja 4, liberando recursos para investir em novos empreendimentos sem comprometer significativamente a receita total da rede.
